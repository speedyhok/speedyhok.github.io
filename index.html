02 / Path

How I got to AI.

Each stage kept the last one's rigor and added a new way to apply it.

Physics

Modeling systems from first principles

Started with the discipline of building a model, deriving a prediction, and checking it against measurement — the same loop that underlies every ML system I build now.

Machine Learning

Learning functions from data instead of deriving them

Moved from closed-form models to learned ones: regression, classical ML, and the statistics of knowing when a model is actually generalizing.

Data Analytics

Uncovering patterns from complex datasets

Applied that quantitative foundation to clean, engineer features, and model structured data—from macroprudential bank risk matrices and security profiles to financial forecasts.

Artificial Intelligence

Shipping full systems, not just models

Now building the layer around the model: ingestion, serving, dashboards, and the UX that turns an inference into something a human can trust and act on.
